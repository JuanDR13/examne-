¿Qué es un sistema gestor de base de datos?
UN SISTEMA GESTOR DE BASES DE DATOS (SGBD) ES UN SOFTWARE PARA  CREAR, ELIMAR, UPDATE, TOTAL PARA ADMINISTRAR UNA BASES DE DATOS. 
COMO FORMA DE GARANTIZAR LA CALIDAD DE DATOS DE X EMPRESA PARA ADMINISTRALA DE LA MEJOR MANERA.

¿De qué se encarga el lenguaje de definición de datos DDL?
EL LENGUAJE DE DEFINICIÓN DE DATOS (EN INGLÉS DATA DEFINITION LANGUAGE, O DDL), ES EL QUE SE ENCARGA DE LA MODIFICACIÓN DE 
LA ESTRUCTURA DE LOS OBJETOS DE LA BASE DE DATOS. INCLUYE ÓRDENES PARA MODIFICAR, BORRAR O DEFINIR LAS TABLAS EN LAS QUE SE ALMACENAN LOS DATOS 
DE LA BASE DE DATOS. EXISTEN CUATRO OPERACIONES BÁSICAS: CREATE, ALTER, DROP Y TRUNCATE. 
Describe al menos tres sentencias que contiene el lenguaje de definición de datos.
¿En qué consiste la sentencia CREATE?
CREATE  ES CREAR
ESTE COMANDO PERMITE CREAR OBJETOS DE DATOS, COMO NUEVAS BASES DE DATOS Y TABLAS.
EJEMPLO (CREAR UNA TABLA)
CRATE TABLE “HERRERIA”;
¿A qué elemento y cuando se aplica la sentencia ALTER?
ALTER | MODIFICAR
ESTE COMANDO PERMITE MODIFICAR LA ESTRUCTURA DE UNA TABLA U OBJETO. SE PUEDEN AGREGAR/QUITAR CAMPOS A UNA TABLA, MODIFICAR EL 
TIPO DE UN CAMPO, AGREGAR/QUITAR ÍNDICES A UNA TABLA ETC
EJEMPLO (AGREGAR COLUMNA A UNA TABLA)
ALTER TABLE “HERRAJES” ADD CALIBRE INT ;

¿En qué consiste la sentencia DROP?
DROP | ELIMINAR
ESTE COMANDO ELIMINA UN OBJETO DE LA BASE DE DATOS. PUEDE SER UNA TABLA, VISTA, ÍNDICE,  FUNCIÓN, PROCEDIMIENTO O CUALQUIER OBJETO 
QUE EL MOTOR DE LA BASE DE DATOS SOPORTE. SE PUEDE COMBINAR CON LA SENTENCIA ALTER.
EJEMPLO DROP TABLE “HERRERIA”;
